Components Used

ESP32

The ESP32 microcontroller acts as the brain of the system, handling data from the load cell and sending real-time monitoring updates.

HX711 Module

The HX711 is a precision 24-bit ADC used to interface the load cell with the ESP32 for accurate weight measurement.

Load Cell

The load cell measures the gas cylinder’s weight, enabling the system to calculate remaining gas levels precisely.

OLED Display

The OLED display shows the live gas level, estimated usage time, and flame mode of the stove
