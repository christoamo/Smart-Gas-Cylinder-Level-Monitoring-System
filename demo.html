How It Works

The system reads the gas cylinder weight in real-time using sensors. Users can select the flame mode, and based on consumption, the system calculates and displays the estimated usage time on the Blynk dashboard. Additionally, it sends a Telegram notification when the gas level drops below 10%, ensuring timely refills and smooth operation.
